**Data Manipulation and Reporting with Power BI**

This project focuses on transforming raw data into meaningful insights using Power BI. It demonstrates the complete data analysis workflow, including data cleaning, transformation, modeling, and interactive dashboard creation.

**Project Overview**

The objective of this project is to analyze datasets and generate actionable insights through visually appealing and interactive reports. Power BI tools such as Power Query Editor and DAX (Data Analysis Expressions) were used to manipulate and model the data effectively.

**Key Features**
Data cleaning and preprocessing using Power Query
Data transformation and relationship building
Creation of calculated columns and measures using DAX
Interactive dashboards with filters and slicers
Visualizations including bar charts, line charts, pie charts, and KPI cards
**Insights Generated**
Identification of key trends and patterns in the dataset
Comparative analysis across different categories
Performance tracking using KPIs
Data-driven decision-making support
